# Personal Portfolio

A clean and responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Sections

- **Home** – Introduction and hero section
- **About** – Personal background and summary
- **Resume** – Education and experience timeline
- **Skills** – Technical skills and proficiencies
- **Projects** – Showcase of personal and academic projects
- **Contact** – Contact form and social links

## Tech Stack

HTML · CSS · JavaScript
